# Windows XP Rice
##### Project's goals: 
* Reproduce **Windows XP** interface as close as possible on **Linux**
* Each part of interface (e.g. "start" menu, bottom bar..) should run independently one from another
* Make it more or less usable and functional (ㆆ_ㆆ)
